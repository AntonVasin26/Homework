1.1) Нужно разбивать большие функции на маленькие подфункции
1.2)добавлять проверку предусловий, трассировка, логирование, проверки на ошибки, проверка постусловий.

К2)В чем заключаеться концепция встраивания встраивания функций
2.1) Вместо того, чтобы вызывать функции при работе программы, сама функция встраивается в код 
	благодаря этому не тратится лишнее время на вызов функции.

К3)Какие аргументы функции могут иметь значение по умолчанию
3.1)Любые главное соблюдать последовательность (по умолчанию идут после переданных)

К4)На основании чего разрешается выбор перегруженной функции

К5)Как обечпечить состояние в функциях и лямбда выражениях
5.1)В лямда функции внешние переменные передаются в квадратных скобках
	[=] передача всех переменных по значению
	[&] всех по ссыслке
	[x, &y]   x- по значению| y- по ссылке
	В обычных функциях есть переменные типа (static)
